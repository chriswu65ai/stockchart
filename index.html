How to use

  • Load a local Excel file containing at least two columns: Date and Price. Optional columns: Event and Comment for annotated bubbles.
  • Scroll or use the timeframe bar to pan through time, or use the presets.
  • Hover event/comment bubbles to view annotation text, and click a bubble when it contains a URL.
  • The Excel file is not uploaded, the file only interacts with the page. All data remains local.
